1\. Navigate to [https://192.168.1.1/](https://10.0.12.1/) or what ever IP does you EdgeRouter X have.

![](https://ajeuwbhvhr.cloudimg.io/https://colony-recorder.s3.amazonaws.com/files/2025-04-22/61b4e363-69ab-4e53-ae67-d1c740c04eee/ascreenshot.jpeg?tl_px=384,49&br_px=3136,1588&force_format=jpeg&q=100&width=1120.0)

![](https://ajeuwbhvhr.cloudimg.io/https://colony-recorder.s3.amazonaws.com/files/2025-04-22/61b4e363-69ab-4e53-ae67-d1c740c04eee/ascreenshot.jpeg?tl_px=360,2&br_px=3112,1541&force_format=jpeg&q=100&width=1120.0&wat=1&wat_opacity=0.7&wat_gravity=northwest&wat_url=https://colony-recorder.s3.us-west-1.amazonaws.com/images/watermarks/FB923C_standard.png&wat_pad=530,106)
